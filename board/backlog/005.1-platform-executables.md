# 005.1 — Platform-aware executable resolution

Extend `ServerInfo` to support per-platform executables (e.g. `PalServer.sh` on Linux). Resolve executable at runtime based on `process.platform`.

**Origin:** Code review — Windows-only `.exe` paths.

## Acceptance criteria

- [ ] `ServerInfo` supports `executable` string or `executables: Partial<Record<NodeJS.Platform, string>>`
- [ ] `run-server`, `stop-server`, `isProcessRunning` use resolved executable
- [ ] Unit tests cover win32 vs linux resolution for Palworld and Enshrouded
- [ ] `npm test`, `npm run type-check` pass
