# 012.1 — Show app version in the UI

Wire the existing `getAppVersion` preload API into the renderer so the installed app version is always visible (e.g. header/footer or about strip). This is the primary way to confirm an auto-update landed.

**Depends on:** none (IPC already exists).

## Acceptance criteria

- [ ] Renderer calls `window.electronAPI.getAppVersion()` (or current typed API) on load
- [ ] Version string is shown in the main UI without navigating elsewhere
- [ ] Displayed value matches `app.getVersion()` / packaged `package.json` version
- [ ] Component or hook test covers successful version render (and a failure/empty fallback if applicable)
- [ ] `npm test`, `npm run lint` pass
