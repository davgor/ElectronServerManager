# 003.1 — Whitelist allowed IPC channels in preload

Restrict `preload.ts` `invoke`/`send`/`on` to a fixed set of channel names. Log or reject unknown channels.

**Origin:** Code review — IPC security.

## Acceptance criteria

- [ ] `ALLOWED_CHANNELS` constant lists every channel registered in `main.ts`
- [ ] Preload throws or no-ops on disallowed channels
- [ ] `src/__tests__/preload/preload.test.ts` covers allowed and rejected channels
- [ ] `npm test`, `npm run lint` pass
